# Final Project Proposal

## Data
For this project, I plan to collect text data from social media posts on Reddit. I will focus on gathering post titles from different subreddits to see what people are talking about.

## Questions
- What are the most common words used in Reddit post titles?
- What topics are discussed the most in a specific subreddit?
- How do the main topics differ between different subreddits?
